Projects

You can find most of the projects I’m working on at GitHub. Some of my recent work include working on the following projects:

  • WildFly (fka JBoss AS): Working on management related subjects like the Admin Console.

  • Schlawiner: Puzzle and dice game based on Errai. Schlawiner is hosted at OpenShift and also available in the Chrome Web Store and the Firefox Marketplace.

  • Karaka: A personal time recording app deployed on Google AppEngine.

  • Piriti: JSON and XML mapper for GWT based on reasonable defaults, a handful of annotations and deferred binding.

Besides I like to contribute to open source projects like GWTP.
